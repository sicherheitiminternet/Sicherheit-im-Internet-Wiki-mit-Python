## 3. Installation / Nutzung

1. Repository herunterladen:  
   Klicken Sie auf „Code → Download ZIP“ oder nutzen Sie Git:  
   `git clone https://github.com/sicherheitiminternet/Sicherheit-Im-Internet-Wiki-mit-Python-Hub.git`

2. Falls Sie das Repository als ZIP-Datei heruntergeladen haben, entpacken Sie diese auf Ihrem Computer.

3. Öffnen Sie die HTML-Datei `index.html` mit einem Browser, um das Wiki zu lesen.

4. **Keine Änderungen oder Weitergabe** vornehmen – die Lizenz erlaubt ausschließlich Lesen.
